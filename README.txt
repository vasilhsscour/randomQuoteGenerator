# randomQuoteGenerator
Random Quote Generator

This page connected with an api (http://quotes.stormconsultancy.co.uk/random.json)
and take the results (author quote and url that quote).

The results put it in the page in a box which is in the middle of the screen.
This box also contain 3 buttons.

The first button is a twitter share button where you can twitte this quote.
The second button is a facebook share button where you can share this quote.
Finally the third button is to take a new quote.

When the user press that button exwpt from the new quote change the background color of the page
and also the color for all the elements in the box.

Also when make the call to this api and take the result the box slide
and slide back again with the new results.
